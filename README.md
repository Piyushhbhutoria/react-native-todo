## Features

1. Add Task
2. Swipe left to delete
3. Mark Completed
4. Pending & Completed Task Views

### Steps to use

```bash
git clone https://github.com:Piyushhbhutoria/react-native-todo.git
cd react-native-todo
expo install
yarn start
```
